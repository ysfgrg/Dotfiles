## Install 

```
git clone https://github.com/ysfgrg/Dotfiles.git ~/.dotfiles
cd .dotfiles
```
- For only The Dotfiles 
```
sh ./install-dots.sh 
```
- For My Suckless Builds
```
sh ./install-all.sh
```
